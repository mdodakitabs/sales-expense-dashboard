# 📊 Smart Sales & Expense Dashboard

A premium, interactive financial dashboard built with **HTML5, CSS3, and Vanilla JavaScript**. It helps track revenue, manage expenses, and visualize financial trends in real-time.

---

## 🔗 Live Demo
✨ **[https://mdodakitabs.github.io/sales-expense-dashboard/]** ✨

---

## ✨ Key Features
- **Dynamic Stats:** Real-time updates for Revenue, Expenses, and Net Profit.
- **Charts:** Interactive visual financial trends powered by `Chart.js`.
- **Smart Filter:** JavaScript-powered instant search and category filters.
- **Local Storage:** Secure data persistence directly in the browser.
- **Premium UI:** Modern Glassmorphism dark theme, fully responsive.
